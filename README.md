Sundown Studio Website Clone

A Frontend Clone of the Sundown Studio Website

Overview

This project is a frontend clone of the Sundown Studio website, built using HTML, CSS, and JavaScript. The goal of this project is to replicate the design, layout, and basic structure of the original website while practicing modern web development techniques.

It focuses on building visually appealing user interfaces and understanding layout structuring, responsiveness, and media integration.

Features
Responsive navigation bar
Structured layout with multiple sections
Embedded video integration
Clean and minimal UI design
Smooth page structure similar to the original website
Tech Stack
HTML5
CSS3
JavaScript
Project Structure
Sundown-Clone/

├── index.html      # Main HTML structure
├── style.css       # Styling and layout

Installation and Setup
1. Clone the Repository
git clone https://github.com/your-AmanBhushanOfficial/sundown-clone.git
cd sundown-clone
2. Open the Project

Simply open the index.html file in your browser:

start index.html

or double-click the file.

Implementation Details
The navigation bar contains links such as Work, Studio, and Contact.
The main section includes a headline and descriptive text aligned in a structured layout.
A background/featured video is embedded using the HTML <video> tag.
The layout is divided into sections for better readability and scalability.
Limitations
This is a static clone (no backend functionality)
Limited interactivity compared to the original website
Animations and advanced effects may not fully match the original
